# wp-SOMAD

컨텐츠 중심의 material wordpresss 테마.

* 커스텀 로고, 단일 사이드바
* [KaTex](https://khan.github.io/KaTeX/)와 [highlight.js](https://highlightjs.org/) 탑재
* 구글 애드센스 페이지 광고 스크립트 지원
* 구글 애널리틱스, 네이버 애널리틱스 스크립트 지원
